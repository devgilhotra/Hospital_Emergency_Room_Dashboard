# Hospital_Emergency_Room_Dashboard
Overview
An interactive Excel dashboard analyzing Emergency Room patient data to help hospital operations managers make better decisions using real KPIs.
KPIs Tracked

Number of Patients — 513 total with daily sparkline trends
Average Waiting Time — 36.32 minutes with daily tracking
Patient Satisfaction Score — 4.96 with daily sparkline
Patient Attend Status — 38% on time, 62% delayed
Admission Status — 52.44% admitted, 47.56% not admitted
Age Group Distribution — patients grouped 0–9 up to 70–79
Gender Analysis — 53% male, 47% female
Department Referrals — General Practice leads with 103 referrals

Tools Used

Microsoft Excel
Pivot Tables
Slicers
Area Sparklines
Pie Charts & Bar Charts
Conditional Formatting
Dashboard Design

Key Insights

62% of patients experienced delays — highlighting workflow improvement opportunities
Age group 0–9 has the highest ER visits (76 patients)
General Practice and Orthopedics are the top referral departments
Daily sparklines connect satisfaction dips to high-volume periods
